# AXIS — SF/추상 축 브레인스톰

5축 status table 위에 얹을 수 있는 SF/추상 축들. 머리에서 짜내는 대로, 중복 직전까지.

## 시공간 · 인과
- **Lightcone-isolation** — 원래 빛원뿔 바깥에서 재현되는가
- **Retrocausal-signature** — 더 이른 커밋/실험에 흔적을 남기는가
- **Tachyonic self-ref** — 미래-자기의 예측을 입력으로 받는가
- **Chronos / Kairos** — 절대시간 기준인가, 무르익은 순간 기준인가
- **Time-crystal phase** — 주기적 시간 결정 위 위상
- **Bardo-liminality** — run과 run 사이 liminal 상태에서도 유효한가
- **Zeno-resolution** — 무한분할 극한에서의 거동
- **Newcomb-tier** — 예측자가 네 선택을 이미 안다는 가정의 강도

## 양자 · 우주론
- **Everett-branch trust** — 어느 Everett 가지의 결과를 신뢰하는가
- **Anthropic-weight** — observer selection bias 보정 후에도 남는가
- **Vacuum-stability** — 현재 진공이 true vacuum이라는 가정
- **Strangelet-risk** — 실행이 진공붕괴를 촉매할 확률
- **Holographic-bandwidth** — 경계면 면적당 비트밀도
- **Boltzmann-brain robustness** — 우연한 관측자에 대해 강건
- **Heisenberg-budget** — 정밀도 × 보존되는 conjugate
- **Decoherence-debt** — 이전 run에서 누적된 잡음세

## 시뮬레이션 · 합의현실
- **Sim-depth tolerance** — "우리는 시뮬레이션 안" 가정을 몇 층 견디는가
- **Reality-tunnel requirement** — 어느 합의현실에 들어가야 참이 되는가
- **Hyperstition coefficient** — 주장됨으로써 자기실현하는 정도
- **Solaris coefficient** — 시스템이 사용자의 기대를 거꾸로 꿈꿔주는가
- **Pascal-mugging audit** — 작은 확률 × 큰 효용에 의해 우선순위가 왜곡됐는가
- **Roko-exposure** — acausal blackmail로부터 숨겨야 하는가
- **Acausal-trade tier** — 다른 우주 협상자와의 거래 산물인가

## 논리 · 메타수학
- **Gödel-tier** — ZFC / large cardinal / unprovable
- **Russell-stability** — 자기참조 하에서 살아남는가
- **Mu-axis** — 질문 자체를 무화시키는가
- **Counterfactual-α** — 미세구조상수가 살짝 다른 세계에서도 성립
- **Necessary vs contingent** — 모든 가능세계에서 참인가
- **Ontological load-bearing** — 거짓일 때 우주가 깨지는가
- **Schelling-stability** — 좌표 없이 다시 같은 점으로 모일 수 있는가

## 문명 · 시간지속
- **Heat-death horizon** — 우주 팽창 몇 e-fold까지 유의미
- **Empire-collapse robustness** — 문명붕괴 후 재구성 가능
- **Linguistic-drift survival** — 100세대 번역을 견디는가
- **Necromantic revivability** — 메인테이너 사후 artifact만으로 복원 가능
- **Singularity-survival** — FOOM 이후에도 읽히는가
- **Memetic fitness** — 유출 시 복제계수
- **Crystalline persistence** — 매체 손실 시 결정구조에 새겨지는가

## 미학 · 존재론

- **Tao-grain** — 결을 따르는가 거스르는가

- **Lovecraft-comprehensibility** — 정신을 유지한 채 이해 가능한가
- **Borges-coordinate** — 무한도서관에서의 책장 좌표

## 기체 · 작가성
- **Substrate portability** — silicon / photonic / neutral-atom / bio / planck-scale
- **Trans-temporal authorship** — 과거-자기 / 미래-자기 / 평행-자기 중 누구의 손
- **Reader-bandwidth** — 이해에 필요한 최소 관측자 대역폭
- **Self-disclosure** — 결과가 자신이 어떻게 도출됐는지 스스로 폭로하는 정도

# AXIS 적용 — hexa-bio 현재 상태에 갖다 댐

대상: F-Q-6-E ramp, LiH 4e/4o, VQE/UCCSD, qmirror replay, hexa-meta lean4 분리.
각 축 → 도발/실험/관찰 한 줄.

## 시공간 · 인과
- **Lightcone-isolation** — 정보적으로 격리된 lab에 4e/4o 회로 보내서 같은 0.001 Ha를 받아오는가. 또는: detached worktree N개에서 random-init VQE 돌려 수렴 원뿔 측정.
- **Retrocausal-signature** — F-Q-6-E 최종 E값이 1f694e8(Ramp A) 시점 artifact에 이미 잠재해 있었는가. "예언 grep" 감사.
- **Tachyonic self-ref** — VQE는 자기 그라디언트로 자기 업데이트. "predictor mode"로 다음 수렴 궤적 사전계산 → 진짜 tachyon 채널인지 캐싱인지 구분.
- **Chronos / Kairos** — 현 CI는 Chronos. Kairos 모드: geometry optimizer가 "이제 준비됨" 신호 보낼 때만 발화.
- **Time-crystal phase** — VQE 파라미터 공간에서 limit cycle 검출. 진동 주기 = 시간결정 위상. 수렴 실패의 새 분류.
- **Bardo-liminality** — qmirror replay 캐시는 run-사이 bardo. 두 번째 run이 첫 번째 부분 그라디언트를 부정 사용할 가능성 audit.
- **Zeno-resolution** — shots ∈ {1e2…1e6} 스케일링 곡선, ∞ 극한에서 수렴 vs 발산 시그니처.
- **Newcomb-tier** — COBYLA 약한 commit, gradient 강한 commit. "예측자가 네 선택을 안다"는 비대칭으로 옵티마이저 정렬.

## 양자 · 우주론
- **Everett-branch trust** — single-shot vs ensemble vs branch-select. trust 정책을 status table 한 칸으로.
- **Anthropic-weight** — 30/30 PASS의 분모는? 시도 run 전체 vs 로깅 run 비율 → publication bias sub-tier.
- **Vacuum-stability** — BO 진공 가정. QED self-energy 켜면 chem-accuracy 유지되는가 — 별 ramp 후보.
- **Strangelet-risk** — sim 0. neutral-atom 하드웨어 ramp 시 Rydberg leakage가 catastrophic — 의무 축으로 승격.
- **Holographic-bandwidth** — 4 qubit / 16 amplitude → ratio 1, holographic saturated. active space 키울 때 ratio 변화 곡선.
- **Boltzmann-brain robustness** — random init과 trained state의 trace-distance. 멀수록 "우연한 일치" 가능성 낮음.
- **Heisenberg-budget** — 그라디언트 component당 shot 예산. 정밀도-잡음 conjugate를 status table에 명시.
- **Decoherence-debt** — sim 0, 칩 day-of-week 의존. calibration drift를 부채로 적분.

## 시뮬레이션 · 합의현실
- **Sim-depth tolerance** — Python → numpy → BLAS → silicon → ?. 4e/4o 결과가 견디는 emulation 층수 명시.
- **Reality-tunnel** — RHF/UHF/ROHF/CASSCF 어느 reference가 "진짜"인가 — 지금은 암묵, 명시화 필요.
- **Hyperstition coefficient** — "LANDED" 라벨 후 수렴 bias 분포 변화. 라벨 전/후 χ².
- **Solaris coefficient** — CASCI를 reference로 박아둔 순간 옵티마이저는 거기로 떠밀린다. reference-free objective ramp 후보.
- **Pascal-mugging audit** — 6e/6o → 8e/8o → … 어디부터 작은 확률 × 큰 payoff에 기만당하는가. 비용/신뢰도 교차점.
- **Roko-exposure** — 양자화학은 benign. 그러나 ASI contributor 평가 (저강도 mugging) 가능성.
- **Acausal-trade tier** — 다른 worktree의 평행-자기가 dual을 한다고 가정한 작업 분할 — 실제 협업 metaphor로 유용.

## 논리 · 메타수학
- **Gödel-tier** — VQE 수렴은 ZFC-내부. "chem-accuracy 달성 가능한가"가 외부 axiom을 요구하는 경우?
- **Russell-stability** — AXIS.md가 status table row가 되면 자기참조. 메타-레벨 분리로 안 무너짐.
- **Mu-axis** — "LiH의 진짜 에너지는?" → BO 자체가 근사라 mu. frame 고정 강요하는 메타-축.
- **Counterfactual-α** — α 변화 → 결합거리 0.01%. 4e/4o energy sensitivity column.
- **Necessary vs contingent** — 4e/4o 선택은 contingent. 6e/6o 이후 origin 망각 위험. 의사결정 보존.
- **Ontological load-bearing** — chem-accuracy False 시 sub-tier 도미노. 우주는 안 깨짐, milestone 그래프는 깨짐.
- **Schelling-stability** — F-Q-6-E 문자선택 임의 → publish 후 Schelling-locked. 명명 안정성 = 거버넌스.

## 문명 · 시간지속
- **Heat-death horizon** — 의미 지속 ~ 한 세기. 양자화학 자체는 영원, 표기는 일시.
- **Empire-collapse robustness** — Python + classical compute만 있으면 재현. 의외로 강건.
- **Linguistic-drift** — 숫자는 살아남는다. "RAMP / LANDED / EXTERNALIZED CLOSURE"는 100세대 안 산다.
- **Necromantic revivability** — README + tests + AXIS.md만으로 부활. AXIS.md가 부활 매뉴얼 일부.
- **Singularity-survival** — pretraining 흡수 확률 ~100%. FAI가 읽음 (이미 읽혔을 수도).
- **Memetic fitness** — 학계-fit / 일반-unfit. 복제계수 ~0.1.
- **Crystalline persistence** — 30KB, quartz etching 후보. AXIS.md는 만년 단위 보존 가능.

## 미학 · 존재론
- **Aesthetic density** — canon@mk1(퇴장)에서 높았다고 추정. hexa-meta 이전 후 재측정 필요.
- **Tao-grain** — 6-fold 모티프 vs 양자화학 자연결(2-fold 스핀, 점군). 거스르는 결 → 마찰 + 흥미 동반.
- **Egregore-stability** — believer 1명, SPOF. AXIS.md가 egregore externalize.
- **Daemon-haunting** — flaky test 있으면 달 위상 / 요일 / 온도 상관 grep.
- **Mythopoetic resonance** — "RAMP B EXTERNALIZED CLOSURE" 매우 강한 신화어. 효과: 동기부여 ↑ / 외부 추적성 ↓.
- **Cthonic depth** — VQE 1 iteration ~5 layer (pyscf/openfermion/cirq/numpy/BLAS). 그 아래는 Lovecraft 영역.
- **Lovecraft-comprehensibility** — 16 amplitude는 머리에 들어간다. gradient landscape는 안 들어간다 — boundary는 active space size.
- **Borges-coordinate** — 다른 hexa-themed quantum repo와 같은 책장. aleph-0 shelf.

## 기체 · 작가성
- **Substrate portability** — 회로 표현 portable, noise model substrate-specific. "abstract gate" / "compiled" 두 표 필요.
- **Trans-temporal authorship** — Ramp 계획서 = 과거-자기, 실행 = 현재-자기. AXIS.md는 과거→미래 자기 채널.
- **Reader-bandwidth** — VQE basics + 약어. 외부인 ramp-up ~2h. AXIS.md는 0으로 못 만듦 (메타-layer 추가).
- **Self-disclosure** — 커밋 메시지 self-disclose, 코드 부분적, AXIS.md 완전 메타.

## 곱(emergent) 축 — 축×축이 새 의미를 낳는 자리

- **Anthropic × Memetic** = 성공 run만 자랑 → 선택압이 밈으로 전파. "publication-as-selection".
- **Retrocausal × Hyperstition** = 미리 박은 LANDED 라벨이 수렴을 끌어당김. self-fulfilling commit.
- **Gödel × Mu** = well-formed but unprovable in frame. BO 안엔 답, 밖엔 mu.
- **Lovecraft × Heisenberg** = 이해하려는 행위가 정밀도를 깬다. observer effect on understanding.
- **Egregore × Necromantic** = 키퍼 사후에도 사는 전통. AXIS.md가 이 곱의 artifact.
- **Solaris × Newcomb** = 옵티마이저가 예상 에너지를 거꾸로 비춤. CASCI-anchored objective의 본질.
- **Daemon × Time-crystal** = 7일/달위상 주기 flaky test. 달력-debug.
- **Hyperstition × Schelling** = 명명 행위가 좌표를 만들어 자기가 자기의 Schelling point가 됨.
- **Holographic × Cthonic** = 경계 정보 / 깊이 구현. 경계만으로 깊이 무시 가능할 때 추상화 성공.
- **Reality-tunnel × Aesthetic** = 어느 reference state를 trust하느냐가 미학적 선택. HF vs DFT의 도덕성.
- **Zeno × Anthropic** = shots ↑할수록 무엇이 살아남는가 — 셀렉션과 수렴의 상호작용.
- **Bardo × Decoherence-debt** = run-사이 캐시 잔류가 잡음세로 이월. 부채는 liminal 공간에 쌓인다.
